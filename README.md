# Expense Tracker

Sistema de controle de despesas desenvolvido em Python utilizando Programação Orientada a Objetos (POO).

## Funcionalidades

- Adicionar despesas
- Atualizar despesas
- Excluir despesas
- Visualizar despesas
- Resumo mensal
- Filtrar por categoria
- Definir orçamento mensal
- Exportar para CSV

## Conceitos utilizados

- Classes e Objetos
- Encapsulamento
- @property
- CRUD
- Tratamento de Exceções
- Manipulação de Arquivos CSV
- Listas e Dicionários

## Como executar

```bash
python expense_tracker.py
```
